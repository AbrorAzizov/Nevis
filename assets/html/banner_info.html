Линейка Hyseac французского аптечного бренда URIAGE - идеальное решение для жирной и комбинированной кожи, склонной к высыпаниям!

В линейке представлены продукты:

  • для очищения (мицеллярная вода, очищающий гель, очищающий крем, функциональное масло);
  • основной уход для комбинированной кожи;
  • для сухой кожи (восстанавливающий крем, маска, тоник);
  • для доброй в анке (крем, сыворотка, со-паста);
  • для автоматической кожи (эмульсия, маска);
  • с spf-защитой (тональный уход, сыворотка).

Эти продукты обеспечивают видимый результат:

  1. сужают поры;
  2. матируют;
  3. уменьшают борьбу с высыпаниями;
  4. восстанавливают кожу.
